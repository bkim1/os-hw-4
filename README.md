## How to run:
1. Must be within the Linux VM given for the class
2. Navigate the terminal to the `src` folder
3. Run `make` 
4. Run `sudo insmod pp4.ko`

## How to remove module:
1. Must be within the Linux VM given for the class
2. Open a terminal
3. Run `sudo rmmod pp4`
